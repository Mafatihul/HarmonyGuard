## Machine Learning

### Technologies
We fine-tune the Transformer model to detect hate comment using TensorFlow and Keras.

### Setup
To setup this project, firstly download dataset in kaggle
```javascript
https://www.kaggle.com/datasets/ilhamfp31/indonesian-abusive-and-hate-speech-twitter-text
```
and then run with google colab or local jupyter notebook.
